使用Alibaba Cloud Toolkit一键发布本地应用到任意环境
2019-11-07|技术

引言

Alibaba Cloud Toolkit （Eclipse 插件，后文简称 Cloud Toolkit）是阿里云针对 IDE 平台为开发者提供的一款插件，用于帮助开发者高效开发并部署适合在云端运行的应用。 您在本地完成应用程序的开发、调试和测试后，可以使用在 IDE （如 Eclipse 或 IntelliJ）中安装的 Cloud Toolkit 插件，通过图形配置的方式连接到云端部署环境并将应用程序快如部署到云端。

使用可以参考：https://help.aliyun.com/product/29966.html?spm=a2c4g.11186623.6.540.4bff4909CBLmFo

环境要求

在 IntelliJ IDEA 中安装和配置 Cloud Toolkit

在 IntelliJ IDEA 中，您可以在 JetBrains 插件市场搜索 Alibaba Cloud Toolkit 并安装。

安装后重启idea，然后 在Settings中可以看到该插件。

可以在设置中设置SSH信息

将Springboot应用发布到本地虚拟机

项目右键，选择Alibaba Cloud->Deploy to Host

然后配置目标机器，要发布到的目录，以及发布完毕后执行的命令。

最后点Run按钮即可。

然后使用Alibaba Cloud View提供的Terminal登录目标机器，查看发布情况。

使用提供的terminal查看应用日志，如果有异常堆栈也可以直接跳转到项目的代码，非常方便。

注意：如果是maven多模块项目，只想发布其中一个模块，在maven注意选择到子模块。

另外，其实执行maven的package命令即可，没必要clean install。

在部署应用后在自带的Terminal打开日志
