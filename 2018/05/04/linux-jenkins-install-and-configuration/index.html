Linux下安装和配置jenkins

本文共计552个字 | 您是第 位看到它们的小伙伴

1.从jenkins官网下载最新的jenkins的war包，地址：https://jenkins.io/.
2.将jenkins.war丢到tomcat中，并启动
3.浏览器输入ip:port/jenkins访问Jenkins。

在启动时，在日志中可以看到jenkins生成了一个管理密码，复制该密码并粘贴到上图中，以解锁jenkins。

4.进入选择插件安装界面，选择第一个（安装推荐的插件）

5.下一步就是创建一个jenkins的管理员


问题：
1.Linux下，将jenkins.war放入tomcat启动报错，但在windows中启动正常

解决：
估计是jdk版本问题，我重新安装和配置了jdk1.8后ok。
