轻量级的java HTTP Server——NanoHttpd
2018-05-11|技术

NanoHTTPD是一个免费、轻量级的(只有一个Java文件) HTTP服务器,可以很好地嵌入到Java程序中。支持 GET, POST, PUT, HEAD 和 DELETE 请求，支持文件上传，占用内存很小。
github地址：https://github.com/NanoHttpd/nanohttpd。
